# Diagrams

This folder contains visual diagrams for the Data Governance & Catalogue Framework.

## Included Diagrams

| Diagram | Purpose |
|---|---|
| governance-operating-model.png | Shows the main roles involved in governing data assets |
| catalogue-lifecycle.png | Shows the lifecycle for registering, defining, approving and maintaining catalogue assets |
| metadata-model.png | Shows how sources, datasets, fields, KPIs, reports and quality rules connect |

These diagrams are designed to make the framework easier to understand for non-technical and technical stakeholders.
