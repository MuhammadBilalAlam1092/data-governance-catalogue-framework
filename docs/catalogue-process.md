# Catalogue Management Process

This document explains how data assets are identified, documented, approved and maintained in the catalogue.

## Purpose

The catalogue process helps organisations move from scattered and undocumented data assets to a trusted and searchable view of their data estate.

The process is intentionally lightweight so it can be used by small data teams, analytics teams and organisations at an early stage of their data governance journey.

Identify → Register → Define → Validate → Approve → Publish → Monitor
